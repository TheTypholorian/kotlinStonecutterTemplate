This is **Vibrancy**, a mod that aims to rework Minecraft's lighting system, while keeping the feel of the game. 
It has raytraced shadows, colored lights, and a bunch of config options, among other features.
If you want to add compatibility with your mod, add or modify block lights, or just learn how the mod works, visit the [wiki](https://github.com/TheTypholorian/vibrancy/wiki).
